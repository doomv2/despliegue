HTML

HTML is a programming language used for the development of Internet pages. It is the acronym that corresponds to HyperText Markup Language, i.e., Hypertext Markup Language.

CSS

CSS is a language that describes the presentation of documents structured in style sheets for different methods of interpretation, ie describes how a document will be displayed on screen, by printer, by voice (when the information is pronounced through a reading device) or in tactile devices based on Braille.
